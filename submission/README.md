# Beeswarm Plot — Salary Distribution Analysis

This repository contains the RAWGraphs Beeswarm Plot created for advanced compensation and salary distribution analysis across departments and experience levels.

## Files Included
- `chart.png` — Beeswarm Plot generated in RAWGraphs (300–512px PNG)
- Input dataset created for analysis (20 rows, representing department, level, and salary)

## Business Context
The visualization helps executives understand pay equity, compensation distribution, and department-level salary variation for strategic workforce planning.
